# alchemy-spring

## Checking out 

To check out the project:

    # Read+Write Access
    git clone git@github.com:zmoog/alchemy-spring.git 

Or 

    # Read-Only Access
    git clone git://github.com/zmoog/alchemy-spring.git 


## Building

Build from source, do the following:

	   cd alchemy-spring
 
(on *nix)

    ./gradlew build

(on Windows):

    gradlew.bat build



# Running on Jetty

To run the webapp on the Jetty web container:

(on *nix)

	./gradlew jettyRun

(on Windows):

    gradlew.bat jettyRun
